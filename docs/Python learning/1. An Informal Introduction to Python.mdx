## 一、数字
1. 加减乘除，除之后向下取整，取余，乘方
> 两个数相除，所得是浮点数

## 二、字符串
1. 单引号，双引号包裹字符串字面量
> 单引号和双引号没有区别
2. \ 表示转义字符，r表示不使用转义
3. print() 输出字符串
> 正常的字符串是不会将转义字符转成对应的含义的，print()会
4. """ 和 ''' 表示多行字符
5. 字符串的拼接，重复
6. 字符串的索引
7. 字符串的切片
8. 字符串的修改
> 字符串是不可修改的，若想修改，需要通过索引，切片和拼接得到新字符串
9. 字符串长度（len()内置函数）

## 三、列表
1. 字面量，用方括号包起来
2. 索引和切片
3. 合并
4. 改变内容（可以通过索引，和切片来改变）
5. list自己的方法：append()在结尾添加新元素
6. 内置函数 len()
7. 列表嵌套
